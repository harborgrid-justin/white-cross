# White Cross Healthcare Platform

## Component Library Documentation

Welcome to the White Cross Healthcare Platform component library. This comprehensive Storybook documentation covers all 162+ components used across the platform.

### About White Cross

White Cross is an enterprise-grade healthcare platform designed for school nurses to manage student health records, medications, emergency communications, and compliance tracking. The platform prioritizes:

- **HIPAA Compliance**: All components handle Protected Health Information (PHI) securely
- **Accessibility**: WCAG 2.1 AA compliant components
- **Dark Mode**: Full dark mode support across all components
- **Responsiveness**: Mobile-first design for all screen sizes
- **Type Safety**: Full TypeScript integration

### Technology Stack

- **Framework**: React 19 with Next.js 15
- **Styling**: Tailwind CSS with custom design tokens
- **Icons**: Lucide React
- **Charts**: Recharts
- **Forms**: React Hook Form with Zod validation
- **State**: Redux Toolkit + TanStack Query

### Design Principles

1. **Healthcare-First**: Components designed for medical contexts
2. **Accessibility**: Keyboard navigation, screen readers, ARIA
3. **Performance**: Optimized rendering and lazy loading
4. **Security**: PHI protection, audit logging
5. **Consistency**: Unified design language

All components follow HIPAA requirements:

- No PHI in localStorage
- Audit logging for sensitive operations
- Session timeout handling
- Secure data transmission
- Access control integration
